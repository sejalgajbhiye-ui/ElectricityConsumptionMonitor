## Brand & Style

Corporate modern visual direction for a data-heavy electricity monitoring dashboard. The layout is card-based, spacious, and optimized for clarity.

## Typography

Uses Inter with large metric displays, compact uppercase labels, and a neutral hierarchy that prioritizes readability.

## Layout

Built around a 12-column desktop dashboard grid with 24px card padding and 16px gutters.

## Components

- KPI cards with large numeric emphasis
- Alert panels for usage spikes and billing threshold warnings
- Consumption history table
- Insights and optimization callouts
- Placeholder chart regions for future analytics integration
